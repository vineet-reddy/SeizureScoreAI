# SeizureScoreAI Slides

Presentation slides built with [Slidev](https://sli.dev/).

## Quick Start

```bash
# Install dependencies
npm install

# Start the presentation (opens browser automatically)
npm run dev
```

The presentation will be available at `http://localhost:3030`

## Controls

- **Arrow keys** or **Space**: Navigate slides
- **o**: Slide overview
- **d**: Toggle dark mode
- **f**: Fullscreen

## Build & Export

```bash
# Build static site (outputs to dist/)
npm run build

# Export to PDF
npm run export
```
